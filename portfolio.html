Automated Attendance

Creating an efficient classroom attendance system using facial recognition.

This project was created in a team of 10 people. My main role was to use OpenCV to detect faces and store them in a database. then when the face was recognized again, it would be used to mark attendance. My other main role was Front-End manager. I led 4 other students that worked on the front-end of the website and ensured that the website was user-friendly and functional.

Tools Used: Python, OpenCV, HTML, CSS, JavaScript

TexAI

Creating a personal AI that can be used by law enforcement in the office and the field.

This is an ongoing project in conjunction with the Coppell Police Dept. in Texas. I am working in a team of 4 people and made great contributions to the front end of the app and create the prototype AI. We have to use a custom LLM trained on law enforcement procedures, so we achieved this by using Ollama to run a pre-trained model in conjunction with RAG training via webscraping.

Tools Used: React, Python, ChromaDB, Ollama
